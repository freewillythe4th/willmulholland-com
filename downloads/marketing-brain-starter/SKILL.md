# Marketing brain (starter)

This folder is the context layer for all marketing work. Before drafting anything, read the files in `context/` so the work starts from our world, not a blank page.

## How to use this folder

1. Read every file in `context/inside/` first. That is who we are, what we sell, and how we sound.
2. Read `context/outside/` next. That is what our market and customers say in their own words.
3. Then do the task. Quote customer language where it fits. Never invent numbers, use only the ones written down here.
4. If something in here contradicts the request, say so instead of silently picking one.

## Rules

- Match the voice in `inside/how-we-sound.md`. When in doubt, plainer.
- Use the exact phrases customers use (from `outside/customer-language.md`) in headlines and hooks.
- If a file is empty or stale, ask for the missing context rather than guessing.

## Growing this brain

Add to it as you go: after every sales call, drop the best lines into `outside/customer-language.md`. After every good piece of writing, note what worked in `inside/how-we-sound.md`. A small brain kept current beats a big stale one.
