# Marketing brain starter

The starter folder from the Intelligent Growth 5-day course. Unzip it, fill in the context files (two minutes each), then point your AI tools at the folder: in Claude, add it to a Project or open the folder in Claude Code / Cowork; in ChatGPT, add the files to a Project.

From then on, every marketing task starts from your business instead of a blank chat.
