Which client do you use?

MCP is the secure connection that lets Claude, ChatGPT and other supported clients use the Intelligent Growth methods. Pick the client you want to work in. You can connect another later.

Add the hosted connector

Open Customize > Connectors in Claude, select +, then choose Add custom connector. Name it Intelligent Growth, paste the URL below and complete the secure sign-in. For Team and Enterprise, an Owner must first add it in Organization settings > Connectors.

Add the hosted MCP in your terminal

Copy this command into Claude Code. Follow the browser sign-in when it opens, then return to the same session.

Add a custom connector

Full custom MCP setup is currently available on ChatGPT web for eligible Business, Enterprise and Edu workspaces. An admin or owner enables developer mode, then opens Settings or Workspace settings > Apps > Create. Add Intelligent Growth, paste the endpoint, choose OAuth and scan the tools.

Use the hosted web endpoint

Add a remote MCP server in your client. If it asks for a server type, choose HTTP, which means it connects to the hosted web address below. Name it Intelligent Growth. If it asks how to sign in, choose OAuth, the secure browser sign-in option.
